실패하면 p(패턴)은 원점으로, 타겟에서의 탐색 위치는 하나 앞으로

```py
p = 'is' # 찾을 패턴
t = 'This is a book~!"
M = len(p)
N = len(t)

def BruteForce(p,t):
    i = 0 # t / 타겟의 인덱스 
    j = 0 # p / 패턴의 인덱스
    while j < M and i < N : # 패턴 내부에 있어야 한다. 
        if t[i] != p[j]:
            i = i-j # -진행횟수 + 1 만큼...
            j = -1 #어차피 내려가면서 +1 되니까... 0 보려구요
        i = i + 1
        j = j + 1 
    if j == M : 
        return i - M  # 성공 #어느자리부터 있는지 알려드리겠읍니다!
    else: 
        return -1 # fail 
```
갔던만큼 빼주고, 하나 더해줌 (타겟)
갔던만큼 빼주고 실행 (타겟)
